Plugin Name: Comm100 Live Chat
Plugin URI: http://www.comm100.com/livechat/wordpressplugin.aspx
Description: This plugin allows to quickly install Comm100 Live Chat on any WordPress website.
Author: Comm100 Live Chat
Version: 1.0
Author URI: http://www.comm100.com/livechat

== Installation ==

1. Upload `wp-comm100-livechat` directory to Wordpress plugins directory (`/wp-content/plugins/`)
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click the 'Live Chat' menu on the left.
4. Register an account or login.
5. Go to the Widgets page to activate live chat widget and set personal customizations for your own live chat.
